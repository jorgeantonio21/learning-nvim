# :help

## Goal
Master ':help' from beginner to expert in focused, repeatable drills.

## Concept
Open help

## Mental Model
Use this command deliberately: one clear intent, one precise action, then return to normal mode awareness.

## Success Criteria
- You can explain when to choose ':help' over close alternatives.
- You can execute it correctly 10 times in a row.
- You can combine it with motion/register/search/window context where applicable.
