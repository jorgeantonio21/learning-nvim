# NVim Command Lessons

One folder per command lesson. Each lesson contains:
- goals.md
- drills.md
- applied.md
- evaluation.md
- nvchad.md

## Naming Rules
- Folder name is the command when filesystem-safe.
- Case-insensitive filesystem aliases:
  - A -> A_upper
  - E -> E_upper
  - F{char} -> F{char}_upper
  - T{char} -> T{char}_upper
  - gu -> gu_lower
  - I -> I_upper
  - N -> N_upper
  - O -> O_upper
  - P -> P_upper
  - R -> R_upper
  - V -> V_upper
  - X -> X_upper
  - ZZ -> ZZ_write_quit
- Slash-based aliases:
  - / -> SEARCH_SLASH
  - ? -> SEARCH_QUESTION
  - :%s///g -> :%s-slash-slash-slash-g
